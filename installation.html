Installation¶

Stable release¶

To install aicssegmentation, run this command in your terminal:

$ pip install aicssegmentation

This is the preferred method to install aicssegmentation, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for aicssegmentation can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/AllenCell/aicssegmentation

Or download the tarball:

$ curl  -OL https://github.com/AllenCell/aicssegmentation/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
