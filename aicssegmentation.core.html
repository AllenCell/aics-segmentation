aicssegmentation.core.hessian.compute_3d_hessian_matrix(nd_array, sigma=1, scale=True, whiteonblack=True)[source]¶

Computes the hessian matrix for an nd_array. This can be used to detect vesselness as well as other features. In 3D the first derivative will contain three directional gradients at each index: [ gx, gy, gz ] The Hessian matrix at each index will then be equal to the second derivative: [ gxx, gxy, gxz] [ gyx, gyy, gyz] [ gzx, gzy, gzz] The Hessian matrix is symmetrical, so gyx == gxy, gzx == gxz, and gyz == gzy. :param nd_array: n-dimensional array from which to compute the hessian matrix. :param sigma: gaussian smoothing to perform on the array. :param scale: if True, the hessian elements will be scaled by sigma squared. :return: hessian array of shape (…, ndim, ndim)
